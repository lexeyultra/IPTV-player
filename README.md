# 📺 Продвинутый IPTV Web Player

Современный, отзывчивый и невероятно удобный веб-плеер для просмотра IPTV-телевидения прямо в браузере или на мобильном устройстве. Проект разработан на стеке React + Vite + Tailwind CSS и спроектирован с упором на эстетичный дизайн и максимальную область просмотра.

---

## ✨ Ключевые возможности

### 📹 Плеер премиум-класса
* **Полностью безрамочный дизайн**: Никаких лишних рамок вокруг видеоплеера — полное погружение в контент.
* **Умная верхняя панель**: На всплывающей прозрачной строке сверху отображается текущий статус и интерактивные элементы управления.
* **Интерактивный Таймер сна**: Перемещен на верхнюю всплывающую строку. Поддерживает быстрое изменение времени кнопками `-5м` и `+5м` прямо во время просмотра с отображением оставшегося времени на экране.
* **OSD (On-Screen Display)**: Мгновенные всплывающие экранные уведомления при изменении громкости, каналов или состояния таймера сна.

### 🗂 Навигация и переключение
* **Полупрозрачное левое меню**: Боковая панель каналов имеет стильный эффект размытия (`backdrop-blur-md`, прозрачность 60%) и легко скользит поверх видео.
* **Удобное скрытие в один клик**: Боковое меню каналов автоматически закрывается при обычном клике по любому месту экрана плеера.
* **Адаптивный каталог под смартфоны**: Кнопки каталога каналов оптимизированы для мобильных устройств — на экранах смартфонов теперь аккуратно отображается **4 колонки в ряд**, обеспечивая компактность и удобную навигацию пальцем.

### ⚙️ Функционал IPTV
* **Импорт плейлистов**: Поддержка добавления плейлистов по прямой ссылке (M3U URL) или вставкой текстового содержимого (M3U Raw).
* **Система Избранного**: Быстрое добавление любимых каналов в один клик (иконка сердечка) для мгновенного доступа в специальной вкладке.
* **Категории и Поиск**: Автоматическая группировка каналов по категориям (из плейлиста) и мгновенный текстовый поиск по названиям.
* **Сканирование каналов**: Встроенная функция проверки плейлиста на доступность потоков (пингование источников перед воспроизведением).

## 💻 Стек технологий

* **Фреймворк**: [React 18](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
* **Сборщик**: [Vite](https://vitejs.dev/) (супербыстрый запуск и HMR)
* **Стилизация**: [Tailwind CSS v4](https://tailwindcss.com/) (плавные адаптивные классы)
* **Анимации**: [Framer Motion](https://www.framer.com/motion/) (физические пружинные анимации меню и OSD)
* **Иконки**: [Lucide React](https://lucide.dev/) (современный минималистичный сет)
* **Медиа-движок**: [Hls.js](https://github.com/video-dev/hls.js/) (для стабильного воспроизведения `.m3u8` потоков в любых браузерах)

---

## 🛠 Запуск в режиме разработки

1. Установите зависимости проекта:
   ```bash
   npm install
   ```
2. Запустите dev-сервер:
   ```bash
   npm run dev
   ```
3. Откройте адрес `http://localhost:3000` в вашем браузере.
